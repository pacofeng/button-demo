# Tooltip

Tooltips are user-triggered messages that provide additional information that briefly explain a page element or function. They can be triggered by mouse/keyboard hover, focus or click.

### Differences

#### 1. Line Height and Letter Spacing:

DLS:

- line-height - small for 14px, medium for 16px, large for 20px

- letter-spacing - small & medium for 0.25px, large for 0.1px

GUI: Default value for line-height and letter-spacing

#### 2. Padding

DLS: Padding 8px 12px

GUI: No padding

#### 3. BoxShadow

DLS: Boxshadow is updated to "0px 7px 10px 0px rgba(0, 0, 0, 0.04), 0px 1px 8px 0px rgba(0, 0, 0, 0.06)"

GUI: Boxshadow "0px 7px 10px 0px rgba(0, 0, 0, 0.04)"

#### 4. Arrow

DLS: Arrow is enabled by default

GUI: Arrow is not enabled by default
