### Differences

#### 1. All the Differences in Text Field

#### 2. Text Field Padding Right (new change after v20230420)

DLS: padding right - 48px for small, 54px for medium, and 66 px for large

GUI: 0 padding right and selected value overlaps with icon

#### 3. Icon

DLS:

- Size: 12px for small, 16px for medium, and 22 px for large Size (new change after v20230420)
- Ripple: no ripple when clicking the icon
- Hover State Background Color: none
- Gap: 8px gap between icons

GUI:

- Size: 24px for all sizes
- Ripple: has ripple
- Hover State Background Color: #eeeff0
- Gap: no gap
