# Tabs

Tabs organise and allow navigation between groups of content that are related and at the same level of hierarchy. Each tab should contain content that is distinct from other tabs in a set. Tabs can be paired with components or nested within components like cards.

### Differences

#### 1. Size: (new change after v20230420)

DLS: different sizes are added to Tabs component:

- font-size - small for 10px, medium for 12px, large for 14px

- height - small for 28px, medium for 32px, large for 36px

- line-height - small for 14px, medium for 16px, large for 20px

GUI: default one size - font-size 16px

#### 2. Border

DLS: border bottom is added

GUI: no border

#### 3. Ripple

DLS: no ripple when clicking the tab

GUI: ripple when clicking the tab
