#### 1. Height: (new change after v20230420)

DLS - 22px/28px/32px height is set based on button size

GUI - height is not aligned with DLS

#### 2. With Icon

DLS: Margin is removed from icon; 12px/16px/16px icon size is set based on button size

GUI: Icon has default marging and icon size from MUI

#### 3. Ripple

DLS: no ripple when clicking the button

GUI: ripple with a different pressed background color

#### 4. Active State Background Color

DLS: background color back to normal after click and cursor move out of the button

GUI: background color keep in active state after click and cursor move out of the button

#### 5. Vertical alignment with/without icon

DLS: buttons with icon and without icon have same vertical alignment

GUI: buttons with icon and without icon have different vertical alignment

#### 6. Disabled State Background Color: (new change after v20230420)

DLS: #d6d8da

GUI: #42688980 - rgba(66, 104, 137, 0.5)

#### 7. Border Radius: (new change after v20230420)

DLS: 4px/6px/8px for small/medium/large button

GUI: 8px for all buttons
