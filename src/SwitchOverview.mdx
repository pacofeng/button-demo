### Differences

#### 1. Size: (new change after v20230420)

DLS: different sizes are added to FormControlLabel component:

- font-size - small for 10px, medium for 12px, large for 14px

- Switch size - small for 12px, medium for 16px, large for 20px

- line-height - small for 14px, medium for 16px, large for 20px

- letter-spacing - small & medium for 0.25px, large for 0.1px

GUI: default one size

#### 2. Disabled State

DLS: text color is #333D47; Switch color is #D6D8DA

GUI: text color is rgba(0, 0, 0, 0.38); Switch color is #13426b

#### 3. Hover State

DLS: no background on hover; Switch color #333D47 is added on hover

GUI: has circle background color rgba(156, 39, 176, 0.04)

#### 4. Ripple

DLS: no ripple when clicking the Switch

GUI: ripple when clicking the Switch
