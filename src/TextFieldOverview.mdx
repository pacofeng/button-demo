# TextField

Text inputs allow users to enter text into a UI with the default heights being 32px. They typically appear in forms and dialogs. Input fields on mobile displays should be at least 14px text size to avoid auto zoom on mobile browsers.

### Differences

#### 1. Size: (new change after v20230420)

DLS: different sizes are added to TextField component

- intextput field height - small for 22px, medium for 28px, large for 32px

- line-height - small for 14px, medium for 16px, large for 20px

- letter-spacing - small & medium for 0.25px, large for 0.1px

GUI: has a different input field height, line-height and letter-spacing

#### 2. Text Field Label Color

DLS: label color #333D47

GUI: label color #5b636b

#### 3. Text Field Label alignment

DLS: align vetically and horizontally center in empty state and filled state

GUI: not center align

#### 4. Text Field Focus State

DLS: border width 1px

GUI: border width 2px

#### 5. Text Field Disabled State

DLS:

- border color #D6D8DA
- no background color

GUI:

- border color #868b8c
- background color #f7f7f8
