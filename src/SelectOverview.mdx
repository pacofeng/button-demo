### Differences

DLS: different sizes are added to select component:

- font-size - small for 10px, medium for 12px, large for 14px

- color: #333D47

- letterSpacing - small for 8px, medium for 12px, large for 16px

- line-height - small for 0.25px, medium for 0.25, large for 0.1px

- height - small for 22px, medium for 28px, large for 32px

GUI:

- font-size - small for 10px, medium for 12px ,large for 14px(the same as DLS)

- color: #5b636b

- letterSpacing: none

- line-height: none

- height - one height 32px
